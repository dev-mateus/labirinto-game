# Labirinto Game

Jogo de labirinto diário, mobile-first, acessível, otimizado para web e PWA.

## Funcionalidades
- Geração de labirintos diários ou aleatórios
- Controle por teclado e toque (mobile)
- Pontuação por tempo de resolução (LocalStorage)
- Histórico dos melhores tempos
- Instalação como PWA (offline)
- Tema claro/escuro
- Design responsivo e acessível (WCAG)
- SEO otimizado

## Estrutura do Projeto
```
labirinto-game/
├── assets/
├── src/
│   ├── css/
│   └── js/
├── index.html
├── manifest.json
├── service-worker.js
└── README.md
```

## Como rodar
1. Clone o repositório
2. Abra o `index.html` em seu navegador
3. Para PWA, hospede em ambiente HTTPS (ex: GitHub Pages)
4. Para instalar como app, acesse pelo navegador no celular e clique em "Instalar App".
